### The Purpose Problem

**Creatures currently do NOTHING except:**
1. Show a discovery modal (one-time dopamine hit)
2. Appear in the Hatchery UI
3. Increment the collection counter (X/184)

**No:** passive bonuses, active abilities, resource generation, board buffs, synergies.

### Legendary Discovery Wall

At tier 4+, legendary chance is only **3%** (weight 3/100).
- Expected discoveries to find all 23 legendaries: **~767 total discoveries**
- Estimated time: **85-130 hours** of focused play
- Players will discover all commons/uncommons quickly, then grind AGES for legendaries

### Biome Egg Problems
- Only 5/16 biomes have shop eggs (Meadow, Forest, Ocean, Enchanted, Celestial)
- No eggs for: Mountain, Sky, Desert, Arctic, Jungle, Swamp, Garden, seasonal
- No rarity guarantees — buying eggs is pure gambling
- No pity timer — can spend 3,300 gems and still miss legendaries

### Biome Gating
- Tier 1: Meadow, Forest, Ocean, Garden
- Tier 2: +Mountain, Sky, Swamp
- Tier 3: +Desert, Jungle, Spring, Summer
- Tier 4: +Autumn, Winter, Arctic
- Tier 5: +Enchanted, Celestial

Arctic at T4 feels arbitrary (should be T3 with Desert/Jungle).

### Top 3 Fixes

1. **Make creatures generate passive gems** (0.5-5/hr by rarity) — instant mechanical purpose
2. **Increase legendary discovery weight from 3% to 13%** — makes completion achievable
3. **Add biome completion rewards** — permanent buffs + secret legendary for completing all 16

### Additional Recommendations

4. Add "Undiscovered Biome Egg" to shop (2x price, guarantees undiscovered creature)
5. Add missing biome eggs for all 16 biomes (80-150 gems each)
6. Add pity timer: guaranteed legendary every 30 discoveries
7. Add creature deployment mechanic (temporary board buffs, 3 daily uses each)
8. Add creature synergy pairs (discover both = unlock bonus)
9. Add rarity-specific eggs (Legendary Egg: 800 gems, guaranteed legendary)
10. Add biome backgrounds that change based on unlocked regions
